#### Build

``` sh
git clone clone git@github.com:alex-courtis/wideriver.git
cd wideriver
make
```

See [CONTRIBUTING](CONTRIBUTING.md)

#### Install / Uninstall

```sh
sudo make install
sudo make uninstall
```
